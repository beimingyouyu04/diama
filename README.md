# 本地优化算法对联邦学习的影响研究
联邦学习作为一种保护数据隐私的分布式机器学习范式，在数据法规日益严格的背景下具有重要的应用价值。本地优化器的选择直接影响到联邦学习系统的准确率，是联邦学习实践中的关键问题。本文系统研究了四种主流优化器（SGD、Adam、AdamW、RMSprop）在联邦学习场景下的性能表现。基于PyTorch框架设计并实现了完整的联邦学习系统，采用狄利克雷分布模拟非独立同分布数据分配，在MNIST和CIFAR-10两个数据集上分别测试了四种优化器在0.01和0.001两种学习率下的收敛性能和最终准确率。实验结果表明：Adam和AdamW对学习率高度敏感，在0.01时表现很差，在0.001时表现优异；SGD对学习率不敏感，具有良好的鲁棒性，在复杂任务CIFAR-10上以82.08%的准确率优于Adam（79.91%）；RMSprop在所有设置下均表现最差，不适合联邦学习场景。本文的研究为联邦学习系统中优化器的选择提供了实验依据和实用建议。

Federated learning is a distributed machine learning paradigm that protects data privacy. As data regulations become increasingly stringent, it holds significant application value. The choice of local optimizer directly affects the accuracy of federated learning systems and is a key issue in practical federated learning applications. This thesis systematically investigates the performance of four mainstream optimizers (SGD, Adam, AdamW, and RMSprop) in federated learning scenarios. A complete federated learning system is designed and implemented based on the PyTorch framework. The Dirichlet distribution is adopted to simulate non-independent and identically distributed data allocation. On two datasets, MNIST and CIFAR-10, the convergence performance and final accuracy of the four optimizers are tested under two learning rates, 0.01 and 0.001. Experimental results show that Adam and AdamW are highly sensitive to the learning rate: they perform poorly at 0.01 but excellently at 0.001. SGD is insensitive to the learning rate and exhibits strong robustness; on the more complex CIFAR-10 task, it achieves an accuracy of 82.08%, outperforming Adam (79.91%). RMSprop performs the worst across all settings and is unsuitable for federated learning scenarios. This study provides experimental evidence and practical recommendations for optimizer selection in federated learning systems.
